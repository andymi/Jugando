Introducción

Bonanza es una herramienta Web para el control de producción de su ganado bovino. Es un sistema desarrollado a medida de las necesidades básicas del productor. Está diseñado para ser intuitivo, con un diseño consistente, fácil de usar para los usuarios. Bonanza está basado en un diseño modular, con niveles de prioridad. Ésta documentación es una guia para el buen uso y comprensión del mismo, explorando los componentes del sistema.

Dispositivos Relacionados

Para llevar a cabo de manera más efectiva las labores diarias del corral, te damos a conocer los dispositivos que fueron implementados para la recolección diaria de los datos:

  • Comedero. Es utilizado para realizar las descargas diarias de las raciones de los ganados vacunos, puede ser programado hasta 7 raciones al dia con sus diferentes horarios y raciones para cada animal lo que lo hace un elemento muy valioso para su corral.
  • Batea. Es otro componente que va de la mano junto con el comedero, se encarga de pesar las raciones que fueron descargadas del comedero, y además la comida restante que quedó en ella una vez que el animal se retire.
  • Lector de RFID.El lector RFID es una herramienta muy innovadora en ésta área, se utiliza para obtener los Dígitos de las etiquetas RFID de cada ganado vacuno.
  • Etiquetas RFID.Las etiquetas RFID son colocadas en las orejas de cada ganado vacuno para ser identificados.
  • Balanza Electrónica. Las balanzas electrónicas se utilizan para pesar de forma diaria o semanal al ganado de manera que se lleve un control exhaustivo del peso de los mismos.

Nota: Si usted no posee estos dispositivos y quiere acceder a ellos comuniquese con el administrador.

Opciones del Sistema

Algunas de las opciones del sistema que podremos encontrar son:

Alta-Baja-Modificación de Registros

Los registros pueden ser cargados de manera manual como por los dispositivos relacionados al sistema.

Control de Stock

El sistema controla el stock de varias materialesInsumos, Materiales del Corral y Ganado Vacuno.

Emisión de Informes

La emisión de informes están pre establecidos en el sistema con los parámetros más comunes de consultas de todo productor.

Componentes

Bonanza está compuesto por roles de usuario, que son clasificados en:

  • Administrador

    • Posee acceso total al sistema. Éste podría ser el mismo dueño

  • Empleado

    • Está limitado en cuanto al acceso de ciertas ventanas del sistema que sólo al administrador tiene acceso. Éste podría ser el contador o capataz de la hacienda.

Soporte entre Navegadores

Bonanza soporta los siguientes navegadores:

  • IE9+
  • Firefox
  • Safari
  • Chrome
  • Opera

Note: IE9 no soporta transiciones o animaciones. Bonanza funciona perfectamente pero sin el uso de animaciones/transiciones en IE9.

Preguntas Frecuentes

Puedo ingresar al sistema sin una cuenta de usuario registrada?

Bonanza requiere que usted se encuentre registrado en el sistema. Para ello debe contactar con el Administrador y que le designe su propia cuenta de usuario.

Necesito capacitación para saber usar el sistema?

No definitivamente, Bonanza fue testeado con personas que no tenían conocimiento alguno del uso del sistema y se sintieron muy cómodos al utilizarlo

Puedo acceder al sistema cuando estoy de viaje?

Afirmativamente puedes acceder sin problema alguno, ya que el servidor local fue configurado para realizar esa acción.
